# Security Policy

## Supported Versions

We focus on securing the latest versions of **Qatrak (قطرك)**. Please ensure you are using the latest release from the Google Play Store.

| Version | Supported          |
| ------- | ------------------ |
| 1.1.0   | :white_check_mark: |
| 1.0.0   | :white_check_mark: |

## Reporting a Vulnerability

As a Computer Science student and security enthusiast, I value the effort of the security community in helping keep my projects safe. 

If you discover a security vulnerability within this project, please **do not open a public issue**. Instead, follow the steps below:

1. **Email me directly:** [ezzeldeen.dev@gmail.com](mailto:ezzeldeen.dev@gmail.com)
2. **Details to include:** - A clear description of the vulnerability.
   - Steps to reproduce (Proof of Concept).
   - Potential impact.

### My Commitment

* I will acknowledge your report within **48-72 hours**.
* I will keep you updated on the progress of the fix.
* I will give proper credit once the vulnerability is patched.
